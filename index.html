GitHub as your database

All data lives in data.json in your GitHub repo. Every browser in the world reads from there — free forever, no server needed.

One-time setup: 1. Go to github.com/new → create a public repo (e.g. mustafa-portfolio)
2. Go to github.com/settings/tokens/new
   • Note: portfolio admin · Expiration: No expiration
   • Tick repo (full repo access) → click Generate token → copy it
3. Paste your GitHub username, repo name, and token below → click Save & push data
4. Done! ✅ Every save you make in admin writes to GitHub automatically.

⚠️ Your token is stored only in this browser's localStorage — never in the HTML or GitHub. Other browsers can read data without it. Only your browser (with the token) can write.
